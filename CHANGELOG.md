# Changelog — МотоТюмень

## 2026-04-05

### Достижения
- Исправлен роутинг `action=achievements` — запрос с `user_id` перехватывался веткой публичных профилей и достижения не загружались
- Исправлена выдача значков вручную — `get_user_from_token` не доставал поле `role` из БД, поэтому проверка `role in ('ceo', 'admin')` всегда была `False`
- Автосинхронизация достижений (`sync_achievements`) теперь корректно срабатывает при открытии профиля

### Уведомления
- Расширен `NotificationContext` — 10 типов событий вместо 5: `friend`, `achievement`, `badge`, `event`, `gymkhana`, `announcement`, `pillion`, `store`, `system`, `admin`
- Настройки уведомлений разбиты по категориям: Профиль, События, Объявления, ZM Store, Система
- Добавлены push-уведомления браузера с запросом разрешения (Notification API)
- Вкладки "Уведомления" и "Звук" объединены в одну страницу с секциями
- Добавлен `pushPermission` стейт и функция `requestPush()`

### Гараж — фото
- Везде где пользовательские фото заменён `object-cover` на `object-contain` с тёмным фоном: карточки гаража, галерея профиля, объявления, `GaragePickerModal`
- Новый компонент `ImageCropper` — кадрирование фото 4:3 при загрузке. Рамку можно тащить мышью/тачем или двигать кнопками-стрелками. Сетка правила третей. Поддержка очереди из нескольких фото
- Исправлена конвертация `dataUrl → File` — заменён асинхронный `fetch(dataUrl)` на синхронный `atob` + `Uint8Array` (файл теперь гарантированно готов до сабмита формы)
- Превью фото в форме добавления/редактирования техники — соотношение `aspect-[4/3]`

### Гараж — модальное окно техники
- Данные о технике (марка, модель, год, объём, мощность) вынесены на фото в виде полупрозрачных пилюль с градиентом снизу
- Детали (пробег, модификации, описание) и кнопки (найти такой же, найти другую, редактировать, удалить) убраны в выезжающую панель **снаружи** модала слева
- Кнопка-триггер панели (синяя, иконка ℹ️) всегда видна на левом крае фото
- Крестик закрытия панели в заголовке панели
- Фото в модале — фиксированное соотношение `aspect-[4/3]`, `object-cover`
- `max-h-[80vh]` на модале — кнопки всегда в зоне видимости

## Структура ключевых файлов

| Файл | Назначение |

| `src/contexts/NotificationContext.tsx` | Уведомления, звук, push, настройки |
| `src/pages/UserSettings.tsx` | Страница настроек пользователя |

### Backend
| Функция | Эндпоинты |
|---|---|
| `auth` | `?action=achievements` — GET/POST/DELETE достижений и значков |
| `auth` | `?action=sync_achievements` — автовыдача достижений по условиям |
| `auth` | `get_user_from_token` — теперь возвращает `role` |

| Таблица | Содержимое |
|---|---|
| `achievements` | 10 достижений (social, garage, activity, time) |
| `user_achievements` | Выданные достижения пользователям |
| `badges` | 5 значков (event, community, expert, role, verified) |
| `user_badges` | Выданные значки пользователям |
